# 2026 デジタル化・AI導入補助金 申請可否チェック

HTML / CSS / JavaScript だけで構成した、GitHub Pages向けの静的Webサイトです。  
過去のIT導入補助金の交付決定状況をもとに、2026年の通常枠・インボイス枠の申請可否を簡易判定します。  
あわせて、`2018-2025採択者一覧.csv` から `2022〜2025年度` の必要項目だけを抽出したJSONで採択者検索もできます。

画面は以下の2タブで切り替える構成です。
- 申請可否チェック
- 採択者検索

検索表示項目は以下です。
- 法人番号
- 事業者名・個人事業主名・屋号
- 年度
- 類型
- 締切回

## ファイル構成

- `index.html`: 画面構造
- `search-data-2022-2025.json`: 2022〜2025年度・必要項目のみの採択者検索用データ
- `style.css`: レイアウト・配色・レスポンシブデザイン
- `script.js`: 質問分岐、状態管理、結果描画、JSON検索
- `2018-2025採択者一覧.csv`: 採択者検索元データ

## ローカル確認

申請可否チェック自体は `index.html` をブラウザで開いて確認できます。  
採択者検索は `search-data-2022-2025.json` を使い、対象年度を `2022〜2025年度` に絞っています。  
初回検索時のみJSON読込と検索用データ準備を行うため、少し時間がかかることがあります。  
検索機能は `fetch()` を使うため、GitHub Pages などHTTP配信環境での確認を前提にしてください。

## GitHub Pages 公開手順

1. この4ファイルをGitHubリポジトリの公開対象ブランチに配置します。
2. GitHub の `Settings` を開きます。
3. `Pages` を開きます。
4. `Build and deployment` の `Source` で `Deploy from a branch` を選択します。
5. 公開したいブランチとフォルダ（通常は `/ (root)`）を選択して保存します。
6. 数分後に発行されたURLへアクセスして表示を確認します。

## 注意

本サイトは簡易チェック用です。  
実際の申請可否は、公募要領や事務局の最新案内を確認してください。
